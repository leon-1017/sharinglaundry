# Deployment

## Build

```bash
npm install
npm run seed:fallback
npm run build
```

If source-site scraping becomes stable later, replace the fallback seed with:

```bash
npm run scrape
npm run build
```

## Output

- Static files are written to `dist/`
- Sitemap output is generated by `@astrojs/sitemap`
- Assets are served from `public/` and copied into `dist/`

## Nginx

```nginx
server {
    listen 80;
    server_name sharinglaundry.com www.sharinglaundry.com;

    root /var/www/sharinglaundry/dist;
    index index.html;

    location / {
        try_files $uri $uri/ $uri/index.html =404;
    }

    location ~* \.(css|js|jpg|jpeg|png|webp|svg|ico)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }

    location = /404.html {
        internal;
    }
}
```

## Notes

- The current form is frontend-only and does not submit to a backend.
- Original route paths are preserved from sitemap-derived data.
- When the third-party form API is ready, wire it into `src/components/InquiryForm.astro`.
