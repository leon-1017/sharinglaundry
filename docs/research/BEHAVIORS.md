# Behaviors

- Header stays fixed at the top with a white background and compact navigation.
- Desktop navigation exposes product categories as a hover dropdown.
- Mobile navigation collapses into a menu disclosure.
- Homepage hero uses a two-column composition on desktop and stacks on mobile.
- Inquiry form is a primary CTA section on the homepage and on detail pages.
- Card surfaces use lift-on-hover behavior through shadow and slight upward translation.
- Footer keeps contact channels and social links visible on every page.
- Static rebuild keeps the form client-side only. Validation happens in-browser with no network submission.
